# Iraq Logistics UK Agents

Static website for the Iraq Logistics UK agent directory.

## Run Locally

Open `index.html` in a browser, or run:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:4173/
```

## GitHub Pages

This site is ready for GitHub Pages. Publish from the `main` branch with source set to the repository root.
